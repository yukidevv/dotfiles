# dotfiles
For 
- bash(.bash_profile,bashrc)
- vim(.vimrc,.vim)
- tmux(.tmux.conf)

## Usage
`$ make`  

```
help                 helpを表示
linux                linuxのため
mac                  macのため
clean_l              linux関連リンクを削除
clean_m              mac関連リンクを削除
list                 Homebrewでインストールしたアプリ一覧
```
